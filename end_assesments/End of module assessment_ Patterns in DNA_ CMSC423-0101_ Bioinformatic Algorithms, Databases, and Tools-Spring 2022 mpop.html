    Assume you are manning a radio telescope trying to detect alien life. The data generated by the telescope is transformed into a series of sounds which are then transcribed into a musical score to make it easier for the operators to interpret it. You are given this score in an electronic format that looks something like: C1 E2 F1 G3 ...  where the letters represent the notes (A-G), and the numbers represent the octave (1-8).  You are asked to write some code that will identify potential alien transmissions.  How would you go about it?

    Briefly describe the outline of a potential algorithm, and the logic behind it. You don't need to write out the actual code for this assignment.

    Your Answer:

    I would create a frequency array of (k-mers/note-mers) and see which occur most frequenty, to be able to analize if there is a pattern (maybe it is a signaling factor kind of like how we can find occurrences of k-mers and it leading to the origin) 

     

    Using the frequency array we take our note-mers that repeat the most and use this to compare with and array holding the signal, of course we now have to use a double sliding window because we have a pitch and also a note associated with the sound signal.  This algorithm would return the positions of where the signal is repeated.  

    aside from the algorithm described above, I would even go as far as comparing those note-mers that occur frequently and comparing with frequencies of human speech to see if they have further meaning.
